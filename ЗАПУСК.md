# Инструкция по запуску AI Copilot v3

## Что нам нужно

- Python 3.9+
- API ключ от Groq (бесплатно на console.groq.com)
- Браузер

---

## Шаг 1 — Установите зависимости

Откройте терминал и выполните:

```bash
cd backend
pip install -r requirements.txt
```

---

## Шаг 2 — Получите API ключ Groq

1. Зайдите на https://console.groq.com
2. Войдите или зарегистрируйтесь (бесплатно)
3. Перейдите в раздел **API Keys**
4. Нажмите **Create API Key** → скопируйте ключ (начинается с `gsk_...`)

---

## Шаг 3 — Создайте .env файл

В папке `backend/` создайте файл `.env`:

```
GROQ_API_KEY=gsk_ваш_ключ_здесь
```

### Windows (PowerShell):
```powershell
$env:GROQ_API_KEY="gsk_ваш_ключ_здесь"
```

---

## Шаг 4 — Запустите backend

```bash
cd backend
python -m uvicorn main:app --reload --port 8000
```

Ожидаемый результат:
```
INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Application startup complete.
```

Проверьте: http://localhost:8000 → должно показать `{"status": "ok", "version": "3.0"}`

---

## Шаг 5 — Запустите демо-сайт

Откройте **новый терминал** (backend оставьте работать!) из корня проекта:

```bash
python -m http.server 3000
```

Откройте: **http://localhost:3000/demo-site/**

---

## Шаг 6 — Тестирование

Нажмите на кнопку в правом нижнем углу и попробуйте:

| Вопрос | Что происходит |
|--------|----------------|
| О чём этот сайт? | Обычный ответ, intent: browsing |
| Сколько стоит MacBook? | Ответ с ценой, intent: interested |
| Есть ли рассрочка? | Ответ про Halyk и Kaspi |
| **Добавь MacBook в корзину** | **Action Engine нажимает кнопку!** |
| **Прокрути к контактам** | **Страница прокручивается!** |

---

## Новые функции v3

### Intent Layer (полоска сверху чата)
- 🔵 **Просматривает** — обычные вопросы
- 🟡 **Интересуется** — спрашивает про товары и цены
- 🟢 **Готов купить** — мигает! высокий шанс конверсии

### Action Engine
Напишите боту "Добавь MacBook в корзину" — он реально нажмёт кнопку!

### Smart Suggestions
При первом открытии бот загружает умные подсказки, основанные на содержимом страницы.

---

## Возможные проблемы

### "GROQ_API_KEY не задан"
Проверьте что `.env` файл находится в папке `backend/`

### Виджет не появляется
Откройте DevTools (F12) → Console → ищите ошибки `[AI Copilot v3]`

### CORS ошибка
Запускайте через `python -m http.server`, а не открывайте `index.html` напрямую
